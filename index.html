Peasy, torrent tracker.

	
		A bittorrent tracker is a server that assists in the
		communication between peers using the bittorrent protocol ...
	
		
	* Peasy is a pure tracker: it offers no content management features.
	  No registration of .torrent files. No authorization schemes. 
	
	* Peasy is fast; an effort is made to make peasy very efficient.
	  Often it will trade consistency for speed. The goal is to handle
	  hundreds of thousands of downloads a day with negligible load.
	  
	* Peasy is scalabale and can run in a distributed environment;
	  the application core is composed of several processes that do
	  not share memory, so each process can run on a different machine;
	  most of this processes can also be instantiated mutiple times
	  on several machines. 
	  
	* Peasy is written in Erlang following the principles of 
	  Erlang Open Telecom Platform (OTP).
